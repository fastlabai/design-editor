## Features

<CardGrid>
  <Card title="Layers + Shapes + Text">
    Layer panel, shape library, custom fonts, stickers, and a typed object-properties bar.
  </Card>
  <Card title="Background Removal">
    Optional in-browser background removal via `@imgly/background-removal` peer dep.
  </Card>
  <Card title="Provider Architecture">
    Bring your own media library, fonts, persistence, and background-removal services.
  </Card>
  <Card title="Next.js Ready">
    Components ship with `'use client'`. Works with App Router out of the box.
  </Card>
</CardGrid>
